# ✨ LIMA ✨

O projeto **LIMA** é uma página web simples desenvolvida com **HTML e CSS**, criada como tarefa do curso. Ele serve para praticar a estruturação de páginas e a estilização de conteúdo.

## 🎯 Objetivo

Aprender a criar páginas web básicas e organizar arquivos HTML e CSS de forma clara.

## ⚡Funcionalidades atuais

- Página HTML com título, parágrafos e imagens.  
- CSS externo para estilizar cores, fontes e layout básico.

## 🚀 Futuras melhorias

O projeto será atualizado com novas funcionalidades e melhorias visuais conforme o aprendizado no curso.
